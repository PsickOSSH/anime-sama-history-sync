Politique de confidentialité

Anime Sama - History Sync | Mise à jour : 05 novembre 2025

1. Données collectées

  • Identité Google : Nom, email, photo de profil (via OAuth)
  • Historique anime : Données depuis localStorage anime-sama.org (visionnage, favoris, watchlist)
  • Tokens : Refresh token et access token pour accéder à Google Drive

Important : L'extension n'accède jamais aux données du serveur anime-sama.org. Elle synchronise seulement les données stockées localement dans votre navigateur.

2. Où vos données sont stockées

Stockage Contenu Chiffrage
Google Drive (appDataFolder) Historique anime (fichier anime-sama-history.json) AES-256 (chiffrement Google)
Navigateur local Refresh token, access token, métadonnées Stockage local non chiffré
Cloudflare Logs techniques (IP, métadonnées renouvellement) Conservés 7-30 jours
Risque de sécurité critique : Le refresh token est stocké en plaintext dans chrome.storage.local ou browser.storage.local. Un attaquant avec accès local (malware, utilisateur partagé avec permissions admin) peut le voler et accéder indéfiniment à votre Google Drive. Ne partagez pas votre ordinateur avec d'autres utilisateurs.

3. Synchronisation multi-appareils

Votre historique est accessible sur tous les appareils où vous connectez le même compte Google. Tout nouvel appareil utilisant votre compte Google peut charger votre historique anime complet. Ne partagez pas votre compte Google si vous souhaitez garder vos données privées.

4. Qui accède à vos données

  • Vous seul : Via votre compte Google ou directement sur Google Drive
  • Cloudflare : Voit les métadonnées de renouvellement de tokens (USA, encadré par clauses contractuelles types RGPD)
  • Google : Stocke vos données, chiffrées

5. Vos droits (RGPD)

  • Accès : L’accès au fichier anime-sama-history.json nécessite l’extension ou un outil compatible avec l’API Google Drive (l’interface Drive Web classique ne l’affiche pas).
  • Suppression : La suppression du fichier n’est pas possible via l’extension. Vous pouvez supprimer le fichier manuellement depuis Google Drive en allant dans Paramètres → Gérer les applications → Anime Sama - History Sync → Supprimer les données associées.
  • Portabilité : Vous pouvez récupérer vos données uniquement via l’extension ou un outil utilisant l’API Google Drive pour accéder au fichier.
  • Retrait du consentement : La déconnexion de l’extension empêche toute nouvelle collecte de données, mais ne supprime pas les données déjà stockées dans Drive. Pour les effacer, utilisez la procédure de suppression via Google Drive décrite ci-dessus.

6. Pas de tracking, pas de pub

  • Aucun cookie
  • Aucun pixel de suivi
  • Aucun analytics
  • Vos données ne sont jamais vendues

7. Sécurité

  • OAuth 2.0 (secrets gérés par Cloudflare, jamais exposés)
  • HTTPS/TLS 1.2+ pour toutes les communications
  • Refresh token valide indéfiniment jusqu'à votre déconnexion ou révocation via votre compte Google
  • Access token renouvelé automatiquement chaque heure
  • Révocation : Si compromis, révoquez manuellement l'accès via Google Account
